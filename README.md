# 🧴 AI Skincare Recommender

## 📌 Project Overview
The AI Skincare Recommender is a smart web-based application that provides personalized skincare routines based on individual user inputs. The system analyzes factors such as skin type, skin concerns, age, and budget to suggest suitable skincare products and routines.
This project aims to simplify the decision-making process for users who struggle to choose the right skincare products from a wide variety of options available in the market.

## 📌 Features
-  Personalized skincare recommendations  
-  Budget-based product suggestions  
-  Intelligent filtering based on skin type and concerns  
-  Simple and user-friendly interface using Streamlit  
-  Data-driven recommendations using a product dataset  

## How It Works
1. The user enters details such as:
   - Skin type (Oily, Dry, Combination, Sensitive)
   - Skin concerns (Acne, Pigmentation, etc.)
   - Age group
   - Budget

2. The system processes the input using Python and filters the dataset.

3. Based on matching conditions, it recommends:
   - Suitable skincare products  
   - A basic skincare routine
